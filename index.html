An Introduction to Shell Scripting

Scientific and engineering programmers sooner or later encounter the command line. Although unfamiliar, the command line offers unprecedented power and precision for even common tasks in data access, formatting, analysis, and processing. This short course is designed to introduce you to the fundamentals of the shell syntax through exercises and case studies.

This course is not an introduction to programming, nor is it intended to replace courses such as CS 101 which introduce programming concepts. We will discuss aspects of basic programming, but this course is strongly oriented towards accomplishing basic practical tasks which science and engineering students and researchers may encounter.

Course Content

We will be working in the EWS Linux computer lab DCL L440. If you want to bring your own computer, please follow these instructions to get set up:

Linux: You're good to go. Just know how to access your terminal.

Mac OS X: You will use Terminal.app, location in your /Applications folder.

Windows: This situation is a little more complicated. The best Windows-based solution is to download and install mSysGit (alternate link), which includes Git Bash, a command-line interface suitable for 90% of the commands we need.
